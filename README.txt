# RuyamiCiz.com Kurulum Rehberi

Bu proje otomatik olarak oluşturulmuştur. Yayına almak için:

1. Bir GitHub hesabı açın ve bu klasördeki dosyaları yeni bir repository'ye yükleyin.
2. Vercel.com'a gidin ve GitHub hesabınızla giriş yapın.
3. "Add New Project" diyerek GitHub'daki bu projeyi seçin.
4. Vercel ayarlarında "Environment Variables" kısmına şunu ekleyin:
   - Name: OPENAI_API_KEY
   - Value: (OpenAI sitesinden alacağınız sk-... ile başlayan anahtar)
5. "Deploy" butonuna basın.

Tebrikler! Siteniz yayında.
